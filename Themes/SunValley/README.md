# Sun Valley Action Center theme for Windows 11 Notification Center Styler

This theme tries to recreate the design of the hidden 10 Action Center UI on Windows 10 builds 18945-18947, along by adding the newer WinUI design to the old Windows 10 AC which includes:
* Rounded corners
* Windows 11-like textboxes, scrollbars, buttons, tooltips, comboboxes, sliders, etc
* Quick action gripper
* Dynamic sized notification panel regarding the amount of notifications
  
**Author**: [Tails](https://github.com/milestprower92)
## Theme selection

The theme is integrated into the mod and can simply be selected from the mod's
settings:

* Open the Windows 11 Notification Center Styler mod in Windhawk.
* Go to the "Settings" tab.
* Select the theme and save the settings.

## Manual installation

The theme styles can also be imported manually. To do that, follow these steps:

* Open the Windows 11 Notification Center Styler mod in Windhawk.
* Go to the "Advanced" tab.
* Copy the content below to the text box under "Mod settings" and click "Save".
